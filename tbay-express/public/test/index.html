tBay API test

Note: tBay API test is based on https://github.com/Harvard-DCE-CSCIE3/cscie31-spring2019/tree/master/09.1-express-rest-api/public.

This HTML/JS page tests the tBay API by listing all; then creating, finding, modifying, and deleting a photo resource using the REST API. A few notes:

  • API endpoints, property names and values for the example tBay app are hardcoded in the client-side /js/api.js file linked in this HTML page.
  • See the console for output as the process completes.
